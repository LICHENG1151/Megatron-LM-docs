# 训练可靠性工程（导航）

> 本目录是《训练可靠性工程》文档集，从 `Megatron-LM-docs` 顶层的单篇「训练高可用与容错」扩展而来。
> **一句话框架**：训练可靠性工程 = **效率（目标）+ 高可用 HA + 容灾 DR + 支撑（可观测性 / 数值 / SRE / 安全）**——「一个目标 + 两支手段 + 一层支撑」。详见 [01](./01-可靠性工程总览与Goodput目标.md)。

## 为什么从一篇拆成一套

原单篇 [高可用与容错](./02-高可用与容错.md) 严格以 Megatron/NVRx **源码为锚**，覆盖在线恢复机制。但千卡万卡的完整可靠性图景**溢出了"高可用"边界**：它还需要**容灾（DR）**，以及一层 HA/DR 都装不下的**支撑工程**（效率 / 可观测性 / 数值稳定 / 电热 / SRE）。于是拆为：概念总览（01）+ 源码锚定的 HA（02）+ 超出框架的工程考量（03）。
