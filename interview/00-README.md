# 面试题集（导航）

> 本目录是一套**独立的面试/自测文档集**，与 `Megatron-LM-docs` 顶层解析文档配套：解析文档负责「讲清楚原理」，本目录负责「考察是否真的掌握」。

## 编号约定

沿用解析文档的命名规范（`<两位序号>[.子序号]-<中文主题>.md` + 统一元信息头），在本目录内独立编号：

| 序号 | 角色 | 对应解析文档中的角色 |
|------|------|----------------------|
| `00` | 本导航 README | `00-README`（仅导航，不对应具体组件） |
| `01` | 提问路线图与分级评分（总纲） | `01-框架透视图解`（全局总纲） |
| `02+` | 各子系统 / 主题题集，**序号尽量对齐所测子系统的解析文档序号**（如并行化=02） | `02`–`11` 各子系统 |

- 每份题集分层组织：**基础概念 → 核心机制 → 组合应用 → 代码阅读 → 代码编写（含完整答案）→ 进阶追问**。
- 每题给出**参考答案**并标注**对应源码位置 / 解析文档**，便于面试官追问、候选人复盘。

## 怎么用（面试官）

先读 [01 · 面试提问路线图与分级评分](./01-面试提问路线图与分级评分.md)：它把下列题库编排成一套**由浅及深、可分级、防背题**的提问流程（分级标尺 L0–L3、5 条能力轴、**能力矩阵表**、漏斗流程、追问阶梯、变条件迁移追问、评分表、按岗位定 Hire Bar）。面试时照路线图走，用题库作弹药。
